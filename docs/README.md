# Dokumentasi DApp Topup SMT

Dokumentasi ini menjelaskan cara menjalankan, memakai, dan mengembangkan DApp topup game berbasis Hardhat, Vue, Metamask, dan token demo `SMT`.

## Daftar Dokumen

- `docs/getting-started.md` — setup awal dan menjalankan aplikasi.
- `docs/user-guide.md` — panduan memakai aplikasi dari sisi user.
- `docs/smart-contracts.md` — penjelasan kontrak `SMTToken` dan `GameTopup`.
- `docs/frontend.md` — struktur frontend Vue dan integrasi wallet.
- `docs/troubleshooting.md` — solusi error umum Metamask/Hardhat.

## Ringkasan Stack

- Smart contract: Solidity + Hardhat.
- Token: ERC20 demo `SMT`.
- Frontend: Vite + Vue 3.
- Wallet: Metamask.
- Network lokal: Hardhat `chainId 31337`.
